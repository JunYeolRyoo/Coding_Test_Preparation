``` python
class Solution:
    def reorderLogFiles(self, logs: List[str]) -> List[str]:
        digitlog = []
        letterlog = []
        res = []
        for elem in logs:
            first_space_index = elem.index(" ")
            if elem[first_space_index+1].isdigit():
                digitlog.append(elem)
            else:
                letterlog.append(elem)
        
        letterlog.sort(key=lambda x: x[:x.index(" ")])  # Sort by identifier first
        letterlog.sort(key=lambda x: x[x.index(" "):])  # Sort by content

        res = letterlog + digitlog
        return res
```

**Time Complexity: O(NL + LMlogM)** <br>
**Space Complexity: O(N)**
