``` python
class Solution:
    def uniquePaths(self, m: int, n: int) -> int:
        res = [[1 for _ in range(n)] for _ in range(m)]

        for r in range(m-2,-1,-1):
            for c in range(n-2,-1,-1):
                res[r][c] = res[r+1][c] + res[r][c+1]
        return res[0][0]
```

**Time Complexity: O(m * n)<br>**
**Space Complexity: O(m * n)**
