``` python
class Solution:
    def subarraySum(self, nums: List[int], k: int) -> int:
        res = 0
        curSum = 0
        hashmap = {}

        for i in range(len(nums)):
            curSum += nums[i]
            x = curSum - k
            if x == 0:  # curSum is equal to k
                res += 1
            if x in hashmap:
                res += hashmap[x]
            if curSum in hashmap:
                hashmap[curSum] += 1
            else:
                hashmap[curSum] = 1
        
        return res
```

**Time Complexity: O(N)**<br>
**Space Complexity: O(N)**
