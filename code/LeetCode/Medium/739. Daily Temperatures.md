``` python
from collections import deque
class Solution:
    def dailyTemperatures(self, temperatures: List[int]) -> List[int]:
        st = deque([0])
        for ind in range(1,len(temperatures)):
            while(len(st)>0 and temperatures[ind]>temperatures[st[len(st)-1]]):
                popedItem = st.pop()
                temperatures[popedItem] = ind - popedItem
            st.append(ind)
        
        for i in st:
            temperatures[i] = 0
        
        return temperatures        
```

**Time Complexity: O(N)<br>**
**Space Complexity: O(N)**
