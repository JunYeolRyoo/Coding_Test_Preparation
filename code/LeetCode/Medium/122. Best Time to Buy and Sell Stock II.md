``` python
class Solution:
    def maxProfit(self, prices: List[int]) -> int:
        res = 0
        buy = prices[0]
        for i in range(len(prices)):
            if prices[i] > buy:
                res += (prices[i]-buy)
            buy = prices[i]
        return res
```

***Time Complexity: O(N)<br>***
***Space Complexity: O(1)***
