``` python
class Solution:
    def findMaxLength(self, nums: List[int]) -> int:
        dic = {}
        curSum = 0
        maxL = 0

        for i in range(len(nums)):
            if nums[i] == 1:
                curSum += 1
            else:
                curSum -= 1
            if curSum not in dic:
                dic[curSum] = i
            
            if curSum == 0 and i+1 > maxL:
                maxL = i+1
            elif i - dic[curSum] > maxL:
                maxL = i - dic[curSum]
        
        return maxL
```

**Time Complexity: O(N)**<br>
**Space Complexity: O(N)**
