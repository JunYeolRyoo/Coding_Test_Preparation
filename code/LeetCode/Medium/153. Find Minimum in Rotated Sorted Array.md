``` python
class Solution:
    def findMin(self, nums: List[int]) -> int:
        l,r = 0, len(nums)-1
        if nums[l] < nums[r]: return nums[0]    # The list was not rotated. Simply return the first element.
        while(l < r):
            mid = (l + r)//2
            if mid == 0:
                l += 1
            else:
                if nums[mid-1] > nums[mid]:
                    return nums[mid]
                elif nums[mid] < nums[len(nums)-1]:
                    r = mid - 1
                else:
                    l = mid + 1
        return nums[l]
```

***Time Complexity: O(logN)*** <br>
***Space Complexity: O(1)***
