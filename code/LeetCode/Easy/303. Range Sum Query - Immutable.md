``` python
class NumArray:

    def __init__(self, nums):
        curSum = 0
        self.sum = []
        for i in nums:
            curSum += i
            self.sum.append(curSum)

    def sumRange(self, left: int, right: int) -> int:
        if left == 0: return self.sum[right]
        
        return self.sum[right] - self.sum[left-1]
```

**Time Complexity: O(N)**<br>
**Space Complexity: O(N)**
