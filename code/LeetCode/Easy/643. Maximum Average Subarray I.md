``` python
class Solution:
    def findMaxAverage(self, nums: List[int], k: int) -> float:
        maxAvg = 0
        curSum = 0

        for i in range(0,k):
            if i == 0: deduct = nums[i]
            curSum += nums[i]

        maxAvg = curSum/k

        for i in range(1, len(nums)):
            if i + k - 1 > len(nums) - 1:
                break
            
            curSum -= deduct
            curSum += nums[i+k-1]
            deduct = nums[i]
            if curSum/k > maxAvg: maxAvg = curSum/k
        
        return maxAvg
```

**Time Complexity: O(N)**<br>
**Space Complexity: O(1)**
